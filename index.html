Projetos em Análise de Dados

Monitoramento de Vagas

Projeto Monitoramento de Vagas com o objetivo de trazer uma base de dados informativa das vagas de Analista, Cientista e Engenheiro de Dados disponíveis no mercado. Dentro desse contexto, considerei vários cenários para análise:

  • Quais posições são predominantes em cada plataforma;
  • Senioridade de acordo com cada posição;
  • Requisitos técnicos e soft skills;
  • Quantidade de trabalhos remotos, híbridos ou presenciais;
  • Quais os benefícios oferecidos em predominância no mercado.

Para a análise desses dados extraídos por webscraping em Python, usei Power Query dentro do Excel para organizar, relacionar e contar quantas vagas possuem determinadas características de beneficios, requisitos técnicos e soft skills. Com essa base pronta, usei o Power BI para relacionar as tabelas e criar os dashboards que mostrassem o cenário atual da área de Dados.

Ferramentas utilizadas

  • Excel, Power Query, tabelas dinâmicas
  • Power BI (DAX, relacionamentos, filtros, visualizações em gráficos de barras, rosca, treemap e cartões)
